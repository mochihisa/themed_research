# 課題研究

## タイトル
三角関数を応用した音声の研究

## 背景
コロナ禍により非対面で正確な情報を多数の人に提供できる人工音声が着目されつつある.  
しかし,既存の人工音声は **"自然な声ではない!!!"** と抵抗感を示す人がいる.

## 目的
自然な音声はどのようなものなのかについて分析し,より様々な用途に活用できるような自然な人工音声を作成する.

## 方法
･人の音声を録画録音しAudacityを用いて周波数の解析(三角関数)を行う  
･解析結果をもとにサイン波の重ね合わせの○○で人工音声を作成する
